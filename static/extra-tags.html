Installation

Installation of the plugin

  1. Install ls-plugin-extra-tags: $ npm add --save-dev ls-plugin-extra-tags
  2. Import it in the index.js file
  3. Use the HTML tags in your presentation

Example for index.js

"use strict"; import SlideshowPlayer from "lecture-slides.js"; import LsPluginExtraTags from "ls-plugin-extra-tags"; window.addEventListener("load", () => { let player = new SlideshowPlayer({ plugins: { HighlightJs: new LsPluginExtraTags({ // These are the default texts. You can leave them out, // if you don't want to change them. labelCarouselNext: "Next Step", labelCarouselPrev: "Previus Step", labelCarouselReset: "Restart" }), } }); player.start(); });

Installation shouldn't be much of a problem if you already got the basic presentation template running. Once your presentation is working, simply follow the steps as outlined on this slide.
