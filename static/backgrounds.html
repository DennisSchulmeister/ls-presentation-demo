This presentation shows just what kind of backgrounds can be used for a slide.

No Background

  • By default slides have no background color by themselves.
  • The background is determined by the presentation theme.
  • Since this slides use the white theme, the background is â€¦ white

Background Color

  • Background colors can be used with data-background-color="..."
  • Everything that works with the CSS background-color setting is allowed.
  • This slide uses an rgb()-code like this: rgb(197, 212, 232)

Background Gradient

  • Gradients and other fancy backgrounds can be used with data-background="..."
  • Everything that works with the CSS background setting is allowed.
  • This slide has a blue gradient: linear-gradient(to bottom, #e1ffff 0%, #b1d8f5 100%)

Background Images 1

  • Background images can be used with data-background-image="..."
  • The image can either be a file or a remote URL
  • The image always covers the complete page background
  • Make sure that the image has a small file size and loads fast

Background Images 2

  • The background doesn't only cover just the slide content.
  • It also covers the explanations at the side
  • The background must therefor be carefully chosen

As you can see, the background also covers the explanations next to the slide. Therefor make sure that your background guarantees a high reading contrast at every place. Test it with as many screen sizes as you can, if it doesn't.
