Code Examples with Syntax Highlighting

How It Looks Like

Python Example

def main(): """ Main procedure of this example program. """ pass if __name__ == "__main__": main()

Here you can see, how a python module can check whether it is merely imported or run as a program. When run as a program, __name__ == "__main__" will be true and the main()-function will be called.

JavaScript Example

function factorial(n) { if (n > 1) return n * factorial(n - 1); else return 1; }

This is what we want to achieve. When you are preparing slides for a programming course you will have many source code examples. With traditional slideshow makers it is quite tedious to colorize the code examples, but with HTML it is dead easy, because the computer can do all the work for us.

The example to the right requires no manual edition whatsoever. As you will shortly see it is simply pasted into the presentation HTML code.

Installation

Installation is as easy as 1, 2, 3:

  1. Install ls-plugin-highlight.js: $ yarn add --dev ls-plugin-highlight.js
  2. Import it in the index.js file
  3. Import one of the stylesheets that come with highlight.js
  4. Use the HTML tags below in your presentation

Example for index.js:

"use strict"; import SlideshowPlayer from "lecture-slides.js"; import LsPluginHighlightJs from "ls-plugin-highlight.js"; import "./node_modules/highlight.js/styles/atom-one-light.css"; window.addEventListener("load", () => { let player = new SlideshowPlayer({ plugins: { HighlightJs: new LsPluginHighlightJs(), } }); player.start(); });

Installation shouldn't be much of a problem if you already got the basic presentation template running. Once your presentation is working, simply follow the steps as outlined on this slide.

Details on step four can be seen on the following slide.

Usage

Simple. Just use the following new HTML tags.

  • <source-code language="..."> for complete code blocks
  • <src-code language="..."> for inline code snippets

For example:

<article> <source-code language="python" filename="example.py"> def main(): pass if __name__ == "__main__": main() </source-code> <p> When the module is nut just imported but run as a program, <src-code language="python">__name__ == "__main__"</src-code> will be <src-code language="python">true</src-code>. </p> </article>

Here you see, how the example from the first slide was made. As can see it is really just two HTML tags which do all the work for you, thanks to the excellent highlight.js library used by this plugin.
